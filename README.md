### Description
Ce projet permet de visualiser des graphs construits a partir d'un flux de donnees.
Vous trouverez 5 graphes differents vous offrant plusieurs vues.
Les graphs ont ete construits avec l'outils **D3JS**

### Lancement du projet
- Clonez le repo.
- Lancer le projet avec un **Serveur Local**.
	- Exemple d'application: *Web Server for Chrome*.
- Rendez-vous à l'adresse indiquée par l'application dans un navigateur (*127.0.0.1:8887*).
- Une fois dans la page d'accueil, utiliser les liens situés en heut de page pour naviguer entre les différents graphs.
